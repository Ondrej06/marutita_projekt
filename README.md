# marutita_projekt
pro otevření hry a webu je nutné mít python a stažené určité knihovny :
Pro WEb: Flask, flask_sqlalchemy, flask_bcrypt, flask_login, datetime
po satžení otevřít a spustit soubor app.py, tím se do terminálu printne odkaz po kliknutí na tento odkaz se otevře prohlížec se stránkou, aby uživatel mohl cokoliv dělat musí se nejříve zaregistrovat či přihlásit. To mu umožní náhled na web s rozdílem zobrazení pro běžného uživatele a pro admina který má více možností co muže na webu provádět. pro ukončení serveru přes flask stačí v terminálu stisknout CTRL + C.
Pro Hru: pygame, math, sys
po satžení otevřít a spustit soubor hra které otevře okno se hrou, pro ukončení ESC a ukončit.
