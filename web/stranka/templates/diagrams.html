Popis hry

O hře

Projectile Combat je rychlá arénová střílečka zasazená do temného vesmírného prostředí. Hráč ovládá malou, ale obratnou loďku a musí čelit vlnám nepřátel, kteří se ho snaží zničit. Každá hra je jedinečná díky náhodně generovaným efektům projektilů a neustále se zvyšující obtížnosti.

Cíl hry

Přežijte co nejdéle! S každým dalším nepřítelem roste vaše skóre a odemykají se nové výzvy. Hra končí, když vám dojde zdraví – pak se všechny statistiky z této hry uloží a vy se můžete porovnat s ostatními hráči na webovém dashboardu.

Ovládání

  • Pohyb: WASD – vlevo, vpravo, skok/létání (mezerník)
  • Létání: Podržte mezerník – spotřebovává výdrž (staminu). Na zemi se výdrž regeneruje.
  • Dash: Stiskněte CTRL + směr (A/D) – rychlý úskok s krátkou dobou nesmrtelnosti. Dash má cooldown.
  • Střelba: Levé tlačítko myši – vystřelí projektil směrem ke kurzoru. Střelba má krátké zpoždění (300 ms).

Projektily a jejich efekty

Každý výstřel má náhodnou barvu, která určuje speciální efekt. Díky tomu je každá hra pokaždé jiná:

  • Fialová (pierce) – průrazný projektil, může zasáhnout více nepřátel za sebou.
  • Zelená (dot) – způsobuje poškození v čase (Damage over Time).
  • Bílá (slow) – zpomalí nepřítele na krátkou dobu.
  • Červená (explosive) – výbušný projektil, při zásahu poškodí i nepřátele v okolí.
  • Žlutá (heal) – při zásahu nepřítele vás nepatrně vyléčí.

Nepřátelé

Nepřátelé se spawnují v pravidelných intervalech po stranách obrazovky a pohybují se směrem k hráči. Jejich rychlost a životy se postupně zvyšují. Každý nepřítel po smrti vytvoří efekt částic a přičte se do statistiky „Zabito nepřátel“.

Zdraví a výdrž

  • Zdraví: Ubývá při kontaktu s nepřítelem (maximálně jednou za 500 ms). Po pádu na nulu hra končí.
  • Výdrž (stamina): Spotřebovává se při létání. Na zemi se rychle regeneruje. Bez výdrže nelze létat.
  • Dash: Během dash jste nezranitelní. Cooldown dash je 1 vteřina.

Statistiky a ukládání

Každá vaše hra je zaznamenána – vidíte datum, počet zabitých nepřátel, kolizí, výstřelů, zásahů a přesnost. Nejlepší výkony se ukládají do vašeho profilu a na webu si je můžete prohlížet a porovnávat s ostatními hráči. Admin má navíc přehled o všech uživatelích a jejich aktivitě.

Jak spustit hru

1. Co budete potřebovat

  • Python verze 3.8 nebo novější – stáhnout zde
  • Příkazový řádek / terminál (na Windows cmd, PowerShell, na Mac/Linux terminál)

2. Instalace knihoven

Otevřete terminál ve složce projektu a spusťte:

pip install pygame flask flask-sqlalchemy flask-bcrypt flask-login requests

Tento příkaz nainstaluje herní engine Pygame, webový framework Flask a další potřebné knihovny.

3. Spuštění serveru (pro ukládání do databáze)

Hra ukládá statistiky do databáze přes webové API. Nejprve spusťte Flask server: 

python web/stranka/app.py

Server poběží na adrese http://127.0.0.1:5000. Ponechte toto okno terminálu otevřené.

Server vytvoří databázi, tabulky a výchozího administrátora (admin@example.com / admin123).

4. Spuštění hry

Otevřete druhý terminál (nové okno) ve stejné složce a spusťte: 

python menu.py

Objeví se herní okno s úvodní obrazovkou.

Bez serveru: Pokud server neběží, hra se normálně spustí, ale statistiky se ukládají pouze do lokálního souboru game_stats.json.

5. Přístup na web

Otevřete prohlížeč a přejděte na:

http://127.0.0.1:5000

Zde můžete:

  • Registrovat nový účet
  • Přihlásit se
  • Prohlížet všechny herní session ostatních hráčů
  • Jako admin mazat uživatele
  • Prohlížet diagramy a popis hry

6. Import starých dat z JSON (volitelné)

Pokud jste hráli dříve bez serveru a chcete staré záznamy nahrát do databáze, spusťte: 

python import_jsonDB.py

Skript načte soubor game_stats.json a odešle všechny záznamy na API. Při startu serveru se tento import provede automaticky a soubor se přejmenuje na game_stats.json.imported.

7. Časté problémy

  • „pip není rozpoznán“ – použijte python -m pip install ... nebo přeinstalujte Python s možností „Add Python to PATH“.
  • Port 5000 je obsazený – v app.py změňte app.run(debug=True) na app.run(debug=True, port=5001) a upravte i base_url v states.py.
  • Herní okno se neotevře – zkontrolujte instalaci Pygame (pip install pygame).

👩‍💻 Pro nového spolupracovníka

Co to je a jak to funguje

Bubble Shooter je dvouvrstvý projekt: Pygame hra (klient) a Flask webový server (backend). Komunikují přes lokální REST API. Databáze je SQLite soubor instance/users.db, který Flask vytvoří automaticky při prvním startu.

Architektura — State Machine

Hra používá návrhový vzor State Machine. Každá obrazovka (Intro, MainMenu, Login, Playing…) je samostatná třída dědící z BaseState. Přechody řídí StateManager v menu.py — zavolá exit() na starém stavu a enter() na novém.

Jak přidat nový typ projektilu

  1. Přidejte barvu do GameConfig.PROJECTILE_COLORS v config.py
  2. Přidejte barvu do Colors v visuals.py
  3. Přidejte mapování barva → "efekt_id" do effect_map v utils.py
  4. Ošetřete efekt v sekci Speciální efekty v metodě update() v states_game.py

Konvence a standardy

  • Dokumentace kódu: docstringy ve formátu Google Style (Args / Returns / Attributes)
  • Každý modul má hlavičkový komentář s popisem účelu
  • Konstanty patří do config.py — žádná magic čísla v logice
  • HTTP požadavky z Pygame běží v Thread — nikdy neblokují herní smyčku
  • Commity pushujte na Git před každou větší změnou

Co projekt řeší

Bubble Shooter propojuje zábavní hru s datovou platformou. Každá odehraná partie se automaticky zaznamenává, statistiky se ukládají do databáze a jsou okamžitě dostupné přes webový dashboard — hráči vidí svůj progres, admini mají přehled o celé platformě.

Technologický základ

Nativní výkon, 144 FPS

Flask + SQLite

REST API, ORM, bcrypt

Bezpečnost

Bcrypt + Sessions

Hashovaná hesla, role

Datová vrstva

SQLAlchemy ORM

1:N, M:N, CASCADE

Neblokující odesílání dat

Frontend

HTML / CSS / JS

Responzivní, dark theme

Klíčové vlastnosti platformy

  • Uživatelské účty — registrace, přihlášení, role (hráč / admin)
  • Automatický sběr dat — přesnost střelby, čas hraní, počet nepřátel, kolize
  • Achievement systém — M:N vztah User ↔ Achievement, automatické odemykání
  • Offline odolnost — při nedostupnosti serveru se statistiky ukládají lokálně a importují po připojení
  • Admin rozhraní — přehled všech uživatelů a session, mazání účtů
  • Škálovatelná architektura — SQLite lze vyměnit za PostgreSQL změnou jednoho řádku v konfiguraci

Možnosti dalšího rozvoje

  • Nahrazení SQLite za PostgreSQL pro produkční nasazení s více uživateli
  • Přidání globálního žebříčku a sezónních turnajů
  • Monetizace přes prémiové kosmetické prvky (skiny projektilů, pozadí)
  • Mobilní port pomocí Pygame CE nebo přepsání do Godot/Unity
  • API rozšíření pro analytický dashboard (grafy, trendy, retence)
  • Cloudové nasazení (Railway, Render, AWS) pro globální dostupnost

Shrnutí

Projekt demonstruje schopnost navrhnout, implementovat a propojit tři nezávislé vrstvy — herní engine, webový backend a databázi — do funkčního celku s čistou architekturou, bezpečnou autentizací a připraveným datovým modelem. Technologický základ je průmyslový standard s minimálními náklady na provoz.

HUD — co vidíte na obrazovce

  • Zelený bar (nahoře vlevo) — zdraví. Při <30 % zčervená.
  • Modrý bar (pod zdravím) — stamina pro létání. Regeneruje se na zemi.
  • Kruh D (vlevo) — cooldown dashe. Zelený = připraven, číslo = zbývající sekundy.
  • Statistiky (vpravo nahoře) — kills, přesnost, výstřely, zásahy, čas.

Typy projektilů a jejich efekty

Barva každého výstřelu je náhodná a určuje speciální efekt:

BarvaEfektTip
● Fialová Pierce — průrazný, zasáhne více nepřátel Ideální do skupiny nepřátel za sebou
● Zelená DoT — poškození v čase (2 s) Střelte a odskočte, nepřítel umírá sám
● Bílá Slow — zpomalí nepřítele na 1,5 s Použijte k úniku, když vás obkličují
● Červená Explosive — výbuch, AoE poškození 50 px Nejsilnější na skupiny, mějte přesnost
● Žlutá Heal — při zásahu nepřítele vás vyléčí Prioritizujte při nízkém zdraví

Tipy pro přežití

  • 🛡 Dash = nezranitelnost — použijte CTRL+A/D ve chvíli, kdy se nepřítel chystá zasáhnout
  • ✈ Létejte úsporně — stamina se regeneruje pouze na zemi; pokud přijdete o staminu ve vzduchu, padnete
  • 🎯 Míření s nadhledem — projektily mají gravitaci, na dálku míříte mírně nad cíl
  • 💛 Žlutá = záchrana — pokud je zdraví pod 30 %, zaměřte se na žluté projektily
  • 📏 Udržujte vzdálenost — kolize ubírá 15 HP každých 500 ms, nepřátelé jsou nebezpeční zblízka
  • 🏃 Pohybujte se — stojící hráč je snadný cíl, střílejte za pohybu
